2.Flex-Wrap

flexWrap enum('wrap', 'nowrap')
flexWrap属性定义了子元素在父视图内是否允许多行排列，默认为no-wrap。

​ no- wrap flex的元素只排列在一行上，可能导致溢出。

​ wrap-flex的元素在一行排列不下时，就进行多行排列。

​ 就是没有包装就直接全部放在一行，如果有包装就将换行。

通常跟着属性是

align-Content 用来将整个放在中间
